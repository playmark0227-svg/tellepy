ブラウザ内で完結。CSV・依頼文・APIキーはサーバーに送信されません。

CSVを読み込む

母集団になるCSVを選びます。ファイルはPC内で処理され、どこにもアップロードされません。

CSVをドラッグ＆ドロップ、またはクリックして選択（複数OK・一都三県なら4県分まとめて）

国税庁 法人番号「全件データ」CSV（ヘッダ無し30列も自動判別・zipは解凍してから） / gBizINFO一括CSV / 手持ちのCSV
⚠ 国税庁サイトの「検索画面からのダウンロード」は上限10,000件です。件数が足りない場合は 全件データのダウンロードから都道府県別zipを取得してください（東京都だけで約110万法人）。
