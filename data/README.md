# ローカル検索用データ（PC内検索・API不要）

このフォルダに CSV ファイルを置くと、gBizINFO API を呼ばずに **PC の中だけ** で
企業リストを検索できます（検索モード「ローカルCSV」）。

複数の `.csv` を置くと全部まとめて検索対象になります。
巨大なファイル（gBizINFOの一括CSVは約1.7GB）でも、1行ずつ読むので大量メモリは不要です。

## 使い方

1. 下のいずれかのCSVをこのフォルダ（`data/`）に置く
2. 管理画面「リスト作成」で検索モードを **ローカルCSV** にして実行

同梱の `sample_companies.csv` で、すぐに動作を試せます。

## 対応するCSVの列名

先頭行（ヘッダ）の列名から自動で対応づけます。以下のいずれかの名前があればOK
（日本語・英語どちらでも可、表記ゆれも吸収します）。

| 内部項目 | 列名の例 |
|---|---|
| 会社名 | 法人名 / 商号又は名称 / 会社名 / name |
| 所在地 | 所在地 / 本社所在地 / 住所 / location |
| 法人番号 | 法人番号 / corporate_number |
| 資本金 | 資本金 / 資本金(円) / capital_stock |
| 従業員数 | 従業員数 / employee_number |
| 業種 | 業種 / industry |
| 郵便番号 | 郵便番号 / postal_code |
| 設立年 | 設立年月日 / 設立年 / founding_year |
| 代表者 | 代表者名 / 代表者 / representative_name |
| URL | URL / 企業ホームページ / company_url |
| 電話番号 | 電話番号 / TEL / phone_number |

- 文字コードは UTF-8（BOM付含む）と Shift_JIS（cp932）を自動判別します。
- 資本金は「円」単位の整数を想定（例: 1000万円 → `10000000`）。
- 所在地の先頭から都道府県を判定します（例: `東京都世田谷区…` → 東京都）。

## どこからデータを入手するか

### 1) gBizINFO 一括ダウンロード（推奨・資本金/従業員数/業種入り）

経産省 gBizINFO の「データダウンロード」から **基本情報CSV（約1.7GB）** を取得します。
資本金・従業員数・業種・設立年などが含まれ、この機能の絞り込みをフル活用できます。

- ダウンロードページ: https://info.gbiz.go.jp/hojin/DownloadTop
- ダウンロードには無料のAPIトークンが必要ですが、**一度落とせば以降の検索はPC内だけで完結**します
  （毎回APIを叩くことはありません）。

### 2) 国税庁 法人番号 全件データ（完全無料・トークン不要）

商号・所在地・法人番号のみ（資本金・従業員数・業種・電話番号は含まれません）。
業種は社名キーワード、地域は都道府県で絞り込めます。属性フィルタは使えません。

- ダウンロードページ: https://www.houjin-bangou.nta.go.jp/download/zenken/
- 都道府県単位でも取得できます（東京都・神奈川県だけ落とすなど）。

### 3) 自分で用意したCSV

上記の列名に沿っていれば、手持ちのリストや他サービスの出力CSVもそのまま検索できます。

> 電話番号は gBizINFO には含まれません。架電まで繋ぐ場合は、電話番号入りのCSV（国税庁データ＋
> 別途エンリッチ、または電話番号付き商用データ）をここに置くのがおすすめです。
